# Weekly LinkedIn Workflow

> Process for creating 3 LinkedIn posts per week.

## Overview

```
CMO Orchestrator (assigns topics)
    ↓
LinkedIn Generator (writes drafts)
    ↓
Gatekeeper (reviews quality)
    ↓
Final posts ready
    ↓
Brand Monitor (tracks performance)
```

## Monday: Planning

**CMO Orchestrator:**
1. Review B-brain/content-calendar.md
2. Assign 3 topics for the week
3. Provide briefs (ICP pain point, strategic goal, angle)

## Tuesday-Thursday: Creation & Review

**LinkedIn Generator:**
1. Read C-core files (especially voice-dna.md)
2. Write 3 drafts
3. Save to O-output/[numbered-folders]/

**Gatekeeper:**
1. Review each draft
2. APPROVE or SEND BACK with specific feedback
3. Save reviews

## Friday: Finalization & Analysis

**Gatekeeper:**
1. Create final-post.md for approved content

**Brand Monitor:**
1. Track previous week's performance
2. Update M-memory/learning-log.md
