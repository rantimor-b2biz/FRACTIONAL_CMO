# Learning Log

> What we've learned about content performance - what works, what doesn't, and why.

---

## How to Use This File

After publishing content, record:
- **Date** - When was this published?
- **Content Type** - LinkedIn post or Article?
- **Topic** - What was it about?
- **Performance** - Engagement metrics (likes, comments, shares, DMs)
- **What Worked** - Why did this resonate?
- **Lesson** - What we'll do differently next time

---

