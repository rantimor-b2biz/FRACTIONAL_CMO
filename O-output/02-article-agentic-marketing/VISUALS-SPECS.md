# Visual Design Specifications - Agentic Marketing Campaign

**Campaign:** "Agentic Marketing Is Not Marketing Automation 2.0"
**Created:** 2026-02-12
**Style:** Minimalist, Executive, B2B Professional
**Brand colors:** Based on rantimor.com brand

---

## Design Principles

### Visual Identity
- **Style:** Clean, minimal, professional
- **Tone:** Executive-level, not flashy
- **Typography:** Sans-serif, readable, authoritative
- **Color palette:** Professional blues/grays with strategic accent color
- **Layout:** Lots of white space, clear hierarchy

### What to AVOID
- ❌ Overly colorful or "Instagram-style" designs
- ❌ Stock photos of people in suits
- ❌ Generic "tech" imagery (circuits, binary code)
- ❌ 3D rendered objects or gradients
- ❌ Too many elements or visual clutter

### What WORKS
- ✅ Simple diagrams and frameworks
- ✅ Clear typography with hierarchy
- ✅ Strategic use of color for emphasis
- ✅ Lots of negative space
- ✅ Contrast-driven designs

## Post #1: Contrarian Hook - "Agentic AI ≠ Automation 2.0"

### Visual Concept
**Type:** Contrasting comparison diagram

**Layout:**
```
┌─────────────────────────────────────────┐
│                                         │
│   Marketing Automation         vs.     │
│   ↓                                     │
│   Executes your strategy                │
│                                         │
│   Agentic AI                            │
│   ↓                                     │
│   Creates strategy                      │
│                                         │
│   The difference matters.               │
│                                         │
└─────────────────────────────────────────┘
```

**Design Specifications:**

**Dimensions:** 1200 x 1200px (LinkedIn square format)

**Background:**
- Clean white or very light gray (#F8F9FA)

**Typography:**
- Header: "Marketing Automation vs. Agentic AI" - Bold, 48pt
- Left side label: "Marketing Automation" - Semi-bold, 32pt, dark gray (#2C3E50)
- Left side text: "Executes your strategy" - Regular, 24pt, medium gray (#5D6D7E)
- Right side label: "Agentic AI" - Semi-bold, 32pt, accent color (#E74C3C or brand red)
- Right side text: "Creates strategy" - Regular, 24pt, accent color
- Bottom tagline: "The difference matters." - Bold, 28pt, dark gray

**Visual Elements:**
- Vertical divider line between two sides (thin, 2px, light gray)
- Arrow pointing down on left side
- Arrow pointing down on right side
- Optional: subtle icon above each (gear for automation, brain for agentic)

**Color Palette:**
- Primary: #2C3E50 (dark gray/blue)
- Accent: #E74C3C (red) or #3498DB (blue)
- Background: #FFFFFF or #F8F9FA
- Text: #2C3E50 (dark), #5D6D7E (medium gray)

**Export:** PNG, 1200x1200px, 72 DPI

## Post #2: Problem Pattern - "The 6-Month Failure Timeline"

### Visual Concept
**Type:** Timeline infographic showing monthly breakdown

**Layout:**
```
┌─────────────────────────────────────────┐
│  The Agentic AI Failure Pattern         │
│                                         │
│  Month 1 → Excitement                   │
│  Month 2 → Activity                     │
│  Month 3 → Quality drops                │
│  Month 4 → CEO questions ROI            │
│  Month 6 → Wrong optimization           │
│                                         │
│  The fix? Clarity first.                │
└─────────────────────────────────────────┘
```

**Design Specifications:**

**Dimensions:** 1200 x 1500px (LinkedIn vertical format)

**Background:**
- White or very light blue gradient (#FFFFFF → #F0F4F8)

**Typography:**
- Header: "The Agentic AI Failure Pattern" - Bold, 44pt, dark
- Month labels: "Month 1", "Month 2", etc. - Semi-bold, 28pt, accent color
- Descriptions: "Excitement", "Activity", etc. - Regular, 22pt, dark gray
- Bottom CTA: "The fix? Clarity first." - Bold, 32pt, accent color

**Visual Elements:**
- Timeline running vertically down left side (dotted line, 3px)
- Circular nodes at each month (filled circle, 16px diameter)
- Each month has short text description to the right
- Gradient from positive (green/blue at top) to negative (red at bottom)
- Bottom section slightly separated with horizontal line

**Color Progression:**
- Month 1-2: Optimistic blue (#3498DB)
- Month 3-4: Warning orange (#F39C12)

## Post #3: Maturity Model - "3 Stages Framework"

### Visual Concept
**Type:** Staircase/progression diagram with 3 stages

**Layout:**
```
┌─────────────────────────────────────────┐
│  Agentic Marketing Maturity              │
│                                         │
│         ┌─────────────┐                 │
│         │  Stage 3    │                 │
│         │  Strategic  │                 │
│    ┌────┴─────────────┘                 │
│    │   Stage 2        │                 │
│    │   Guided Agents  │                 │
│  ┌─┴──────────────────┘                 │
│  │  Stage 1           │                 │
│  │  Reactive          │                 │
│  └────────────────────┘                 │
│                                         │
│  Most startups: Stage 1 → Stage 3 ❌    │
│  Smart path: Stage 1 → Stage 2 → 3 ✅   │
└─────────────────────────────────────────┘
```

**Design Specifications:**

**Dimensions:** 1200 x 1200px (LinkedIn square format)

**Background:**
- Clean white (#FFFFFF)

**Typography:**
- Header: "Agentic Marketing Maturity" - Bold, 42pt, dark
- Stage labels: "Stage 1", "Stage 2", "Stage 3" - Bold, 32pt, white text
- Stage names: "Reactive", "Guided Agents", "Strategic" - Semi-bold, 24pt, white text
- Bottom text: Regular, 20pt, dark gray
- Checkmarks/X marks: 28pt

**Visual Elements:**
- 3 stacked boxes creating staircase effect
- Each box is a different shade of same color (gradient from light to dark)
- Stage 1 (bottom): Light gray or light blue
- Stage 2 (middle): Medium blue
- Stage 3 (top): Dark blue or accent color
- Arrow showing progression from Stage 1 → 2 → 3
- Red X next to "Stage 1 → Stage 3"
- Green checkmark next to "Stage 1 → Stage 2 → 3"

**Color Palette:**
- Stage 1: #AED6F1 (light blue)
- Stage 2: #5DADE2 (medium blue)
- Stage 3: #2874A6 (dark blue)
- Accent: #E74C3C (red for X), #27AE60 (green for checkmark)

**Export:** PNG, 1200x1200px, 72 DPI

## Tools for Creating Visuals

### Recommended Tools:

1. **Canva** (easiest, templates available)
   - Use "LinkedIn Post" template (1200x1200)
   - Professional, clean templates
   - Export as PNG

2. **Figma** (for custom designs)
   - Full design control
   - Collaborative if working with designer
   - Export at 2x for retina displays

3. **Adobe Express** (formerly Spark)
   - Quick social media graphics
   - Professional templates

4. **PowerPoint/Keynote** (surprisingly effective)
   - Set slide size to 1200x1200px
   - Use brand colors and fonts
   - Export as high-res PNG

### If Hiring a Designer:

Provide them with:
- This spec document
- Brand colors from rantimor.com
- Font preferences (if any)
- Examples of styles you like

## Quick Start: DIY in Canva

### For each post:

1. **Open Canva** → "LinkedIn Post" (1200x1200px)

2. **Choose template:**
   - Search: "minimal professional LinkedIn"
   - Pick clean, simple templates
   - Delete stock photos/unnecessary elements

3. **Apply design specs:**
   - Copy layout from specs above
   - Use specified typography sizes
   - Apply color palette

4. **Export:**
   - Download as PNG
   - High quality (300 DPI if available)

5. **Save in campaign folder:**
   - Name: `visual-post-01.png`, `visual-post-02.png`, etc.
